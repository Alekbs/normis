# О проекте

Проект представляет собой **электронный дневник для детской музыкальной школы**.  
Он позволяет вести расписание групповых и индивидуальных занятий, управлять оценками учеников и отслеживать посещаемость.  
Реализованы личные кабинеты для преподавателей, учеников и завуча с разными правами доступа.  
Есть функционал подачи документов, записи на прослушивания и утверждения расписания.  
Также поддерживаются **реaltime-чаты** для общения между участниками через Centrifugo.

# Запуск проекта

Для запуска сервера используйте команду:  
```bash
dotnet run
centrifugo -c config.json
```
